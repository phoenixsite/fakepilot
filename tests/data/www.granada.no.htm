About Granada AS

Information provided by various external sources

Ønsker du å selge varer via en nettbutikk? Om du skaffer varene kan Granada gjøre resten. Granada tilbyr lagring, plukk/pakk og returtjenester for nettbutikker og andre bedrifter. Vi har en full-service kundesenter som håndterer telefoner, mail, chat, samt henvendelser som kommer via sosiale medier. Granada oversetter tekster som nyhetsbrev, produktinformasjon og nettsider. Vi kan jobbe direkte i ditt ordresystem eller PIM system. La oss ta vare på dine kunder og dine utsendelser, så får du tid til andre oppgaver som markedsføring og salg.
